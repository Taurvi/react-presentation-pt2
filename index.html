What is React.js?

  • Built by Facebook and Instagram

  • Javascript Library - Builds User Interfaces

  • Uses an awesome Virutal DOM

Why React.js?

  • Alternative to Angular (View Only)

  • Can modify pages without refresh

  • Very easy to use

How to use React.js?


// Generic Variable
var toWhom = 'World!';

// Create an Class
var NewApp = React.createClass({
	render:function() {
		// Refer to variables with {}
		return <h1>Hello {toWhom}</h1>
	}
});

// Render Class
React.render(<NewApp />, document.body);
